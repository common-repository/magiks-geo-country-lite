What do you get in your zip file?

  1. In the /magiks_geo_lite/ folder you will find the script file(s).
  2. In the /magiks_geo_lite/license/ folder you will find the GPL license and MaxMind database license files.
  3. In the /magiks_geo_lite/GeoIP/ folder you will find the binary database (from MaxMind) and any supporting script files.
  4. In the /magiks_geo_lite/flags/ folder you will find both the small and large versions of the country flag images.
  5. This readme.html file is supplied also as your overall guide.

Preinstall instructions

  1. You should have downloaded this free plugin from GeoTargetingScripts.com ( or WordPress.org ) and unzipped the files if you are reading these instructions. If you do not have the plugin you can download it here.
  2. This plugin requires PHP version 5. Learn how to find out what PHP version you are using here. If you are currently using PHP <5 and if your host supports PHP5 and the following upgrade method, you should be able to quickly upgrade your hosting to PHP5 by adding the following statement to your .htaccess file: AddHandler x-httpd-php5 .php (click here for more information).
  3. Be sure you upload the unzipped files to your server in "binary" mode when using your FTP software (it is the database that primarily requires binary mode for uploading). Click here for more information. You may also upload the entire zip file first, then unzip it on the server.

Installing and using the WordPress plugin

  1. Upload the entire contents of the 'magiks_geo_lite' folder (including the folder itself) to the /wp-content/plugins/ folder in your WordPress installation.
  2. Login to your WordPress account as an administrator and 'Activate' the 'MaGiKS Geo Country Lite' plugin on the Plugins page (/wp-admin/plugins.php).
  3. You can now display country-targeted messages using WordPress shortcodes. The shortcodes can be used in posts, pages and widgets. Additionally, with a bit of formatting (shown below), the shortcodes can be used in template/theme files as well.
  4. To simply display the country name or show the country's flag you can use the defined tokens as well as the [m_quickgeo] shortcode. Here are the tokens that can be used:

    {flag-large} - displays a larger flag of the visitor's country (image width: 80px, height: varies slightly for each flag)
    {flag-small} - displays a smaller flag of the visitor's country (image width: 16px, height: 11px)
    {country} - displays the 2-character country code (i.e. CA)
    {country-name} - displays the name of the country (i.e. Canada)
    {ip-address} - displays the IP Address of the visitor
  5. As an example, if you run an e-commerce site that ships worldwide, you can use tokens in the [m_quickgeo] shortcode like so:

    In posts/pages/widgets:
    [m_quickgeo]{flag-large} Shipping directly to {country-name}![/m_quickgeo]
    In your template:
    <?php
if (class_exists('magiks_geo_country_shortcode_class'))
{
   echo do_shortcode('[m_quickgeo]{flag-large} Shipping directly to {country-name}![/m_quickgeo]');
}
?>

Database upgrade

This product includes GeoLite data created by MaxMind, available from http://www.maxmind.com/.

Although not necessary, a new version is available for download each month. The compressed version can be found here: http://geolite.maxmind.com/download/geoip/database/GeoLiteCountry/GeoIP.dat.gz. After unzipping (using the RAR utility, you can unzip the compressed database, which is in .gz format), you can replace the database file: GeoIP.dat found in: /magiks_geo_lite/GeoIP/ (be sure to upload the database in "binary" format, click here for more information).
